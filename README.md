# 🔐 Cyber Threat Intelligence System

## 🚀 Overview

This project simulates multiple organizations detecting cyber threats using Machine Learning and sharing them securely using Blockchain (Ganache), all running in Docker containers.

---

## 🧠 Architecture

ML Model → Detect Threat
↓
Blockchain (Ganache) → Store Alerts
↓
Streamlit Dashboard → Visualize Threats

---

## ⚙️ Technologies Used

* Python
* Machine Learning (Scikit-learn)
* Blockchain (Ganache, Solidity, Web3.py)
* Docker & Docker Compose
* Streamlit (Frontend Dashboard)

---

## 🐳 How to Run

### 1. Install Docker Desktop

### 2. Run the project

```bash
docker compose up
```

### 3. Open Dashboard

http://localhost:8501

## 💡 Features

* Real-time threat detection
* Blockchain-based secure logging

Organizations can securely share cyber threat intelligence without trusting each other.
